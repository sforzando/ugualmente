# ugualmente

## How to Develop

```shell
$ npm install
```

## How to Preview

```shell
$ npm start
```

## How to Build MacOS Application

```shell
$ npm run build
```

`build/ugualmente-1.0.0.dmg` is the installer.
Mount it and deploy `ugualmente.app` to `/Applications`.

## How to Launch

### Requirements

* [cliclick](https://www.bluem.net/de/projekte/cliclick/)
  * `brew install cliclick`
